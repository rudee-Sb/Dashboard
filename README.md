# Personal Student Dashboard
A personalized React dashboard with custom widgets and tools like Pomodoro, notes, and tasks—built for my daily productivity.

---

## 🚀 Features

- *Pomodoro Clock* – Boost focus with a simple productivity timer.
    
- *Notes App* – Create, edit, and manage your personal notes.
    
- *Weekly Task Tracker* – Track and organize tasks week by week.
    
- *GitHub Contribution Graph* – View your GitHub activity visually.
  
- *Stat Boxes* – Display quick stats like tasks done or focus time. 

- *Clock & Sticky Notes* – Real-time clock and quick jot notes.

---

## 🧩 Technologies Used

- *React* – Core library for building the UI.

- *Material UI (MUI)* – For UI components and icons.

- *React Pro Sidebar* – To build a responsive, collapsible sidebar.

- *React Router* – For managing navigation between dashboard scenes.

- *react-github-calendar* – To display GitHub contribution data.
